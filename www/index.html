Course structure

1/ Introduction to data analytics

  • What is data analytics?
  • Difference between data analytics and data science
  • Different types of data analysis
  • Some data analytics real-world case studies
  • What does a data analyst do?
  • What is the typical process that a data analyst will follow?
  • Tools and techniques do data analysts use

2/ Statistics for data analysis

  • Statistics from basics
  • Descriptive Statistics - Mean, Mode, Median, Standard Deviation, Variance and Interquartile Range
  • Probability Distributions - Normal, Binomial and Poisson
  • Statistical tests

3/ Data analysis in Excel

  • Preparation of data and data cleaning
  • Creating flexible data aggregations using pivot tables
  • Representing data visually using pivot charts
  • Calculating margins and other common ratios using calculation on pivot table
  • Filtering data using slicers in multiple pivot tables
  • Creating aggregate reports using formula based techniques

4/ Working with Google Data Studio, Power BI and Tableau

frequently asked questions

Is the program 100% online?

Yes, the program is entirely online, so you can study when and wherever you’d like. At the same time, the program is filled with a lot of interaction with your teacher and other students, so you definitely won't feel lonely.

Can I study only one module that I am most interested in?

The course is a complete system, specially designed to become a highly professional data analyst. Viewing the course in parts will not give the desired effect, so this is not possible.

Are there some requirements for the job guarantee?

There are a number of conditions that graduates must meet in order to receive a job guarantee. You must complete 100% of the course and complete all assignments, complete the free Job Ready course, follow the recommendations of career counselors, and submit at least 5 job requests per week during 6 months.

How many hours per week do I need to work on this course?

Typically, the learning process and assignments take from 4 to 10 hours a week.
