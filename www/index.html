2/ Statistik untuk seorang Data Analyst

  • Statistik dari dasar-dasar
  • Statistik Deskriptif - Rata-rata, Modus, Median, Standar Deviasi, Varians dan Jangkauan Interkuartil
  • Distribusi Probabilitas - Normal, Binomial dan Poisson
  • Tes statistik

3/ Data analysis dalam Excel

  • Persiapan data dan pembersihan data
  • Membuat agregasi data yang fleksibel menggunakan PivotTable
  • Merepresentasikan data secara visual menggunakan diagram Pivot
  • Menghitung margin dan rasio umum lainnya menggunakan perhitungan pada PivotTable
  • Memfilter data menggunakan Slicer di beberapa PivotTable
  • Membuat laporan agregat menggunakan teknik berbasis rumus.
