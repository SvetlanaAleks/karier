Struktur kursus

Kau tidak akan menemukan cara lebih cepat untuk menjadi Data Analyst

1/ Data Analytics

  • Data Analyst itu siapa dan apa yang mereka lakukan.
  • Perusahaan mana yang membutuhkan Data Analytics.
  • Keterampilan apa yang dibutuhkan untuk menjadi Data Analyst.

2/ Metrik bisnis utama

  • Apa itu metrik bisnis.
  • 4 KPI bisnis utama.
  • Bagaimana melacak metrik.

3/ Statistik untuk Analisis Data

  • Apa itu statistik dan probabilitas.
  • Bilangan acak. Histogram dan grafik persegi panjang.
  • Distribusi data. Pengukuran distribusi.
  • Analisis korelasi dan distribusi data umum.
  • Pengujian hipotesis.

4/ Analisis data di Excel

  • Pengantar Excel: Fungsi.
  • Rumus, fungsi, dan diagram utama.
  • Pecahkan kasus bisnis dengan bantuan Excel.

5/ SQL dan bekerja dengan database

  • Pengantar SQL: definisi, tujuan dan penggunaan.
  • Instruksi dasar SQL: menulis kueri di database dengan menggunakan SQL.
  • Pecahkan kasus bisnis dengan bantuan SQL.

6/ Otomatisasi analitik menggunakan Python

  • Instalasi Python dan Jupyter.
  • Dasar-dasar pemrograman (Variabel, Operasi, Pernyataan Bersyarat, For loops).
  • Fungsi dan struktur data di Python.
  • Data Mungling. Visualisasi data.
  • Uji statistik.

7/ Visualisasi data dengan Google Data Studio, Power BI, dan Tableau

  • Pengantar Visualisasi data: definisi, tujuan dan penggunaan.
  • Pilihan konsep visual yang efektif.
  • Membuat visualisasi data dengan bantuan Tableau.
  • Membuat visualisasi data dengan bantuan Google Data Studio.

8/ Aplikasi praktis dari analisis data untuk pemasaran dan penjualan

  • Corong akuisisi pelanggan.
  • Analisis pelanggan dan aplikasi.
  • Segmentasi pelanggan dengan bantuan analisis RFM.

9/ Draf proyek akhir dan pembuatan portofolio

  • Rumuskan tujuan bisnis dan pilih metrik yang cocok.
  • Membuat kesimpulan yang terfokus pada pemecahan masalah.
  • Siapkan presentasi bisnis yang jelas.

9/ Karir di bidang analisis data dan persiapan pencarian kerja

  • Perekrutan.
  • CV dan surat lamaran kerja.
  • Pertanyaan umum wawancara kerja.
