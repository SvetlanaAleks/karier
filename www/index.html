Struktur kursus

Kau tidak akan menemukan cara lebih cepat untuk menjadi Data Analyst

1/ Introduction to data analytics

  • Who data analysts are and what do they do.
  • What companies need data analysts.
  • What skills needed for becoming a data analyst.

2/ Key business metrics

  • What are business metrics.
  • 4 main KPIs in business.
  • How to track the metrics.

3/ Statistics for data analysis

  • What is statistics and probability.
  • Random Variable. Histogram and boxplot.
  • Data distribution. Distribution measurements.
  • Correlation analysis and common data distribution.
  • Hyphotesis Testing.

4/ Data analysis in Excel

  • Intro to Excel: Feature.
  • Basic Formulas, functions, and charts.
  • Solve business case using Excel.

5/ SQL and working with databases

  • Intro to SQL: Definition, Goal, and Use.
  • Basic SQL Statements: write database queries using SQL.
  • Solve business case using SQL.

6/ Automation of analytics using Python

  • Python and Jupyter Installation.
  • Programming basics (Variable, Operations, Conditional Statement, For loops).
  • Functions and Data Structure in Python.
  • Data Mungling. Data Visualization.
  • Statistical test.

7/ Data visualization with Google Data Studio, Power BI? and Tableau

  • Intro to Data Visualization: Definition, Goal, and Use.
  • Selection of effective visual concepts.
  • Create data visualization using Tableau.
  • Create data visualization using Google Data Studio.

8/ Practical application of data analytics to marketing and sales

  • Customer acquisition funnel.
  • Customer Analytics and applications.
  • Customer Segmentation with RFM analysis.

9/ Final project and creation of portfolio

  • Formulate a business problem and select relevant metrics.
  • Draw problem-oriented conclusions.
  • Prepare a clear business presentation

9/ Career in data analytics and preparation for job search

  • Hiring Process.
  • CV and Cover Letter.
  • Common interview questions.
